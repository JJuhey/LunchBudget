# Lunch Budget
## GOAL
예산에 따른 점심값을 계획적으로 사용할 수 있도록 간단하게 설정한 예산 대비 사용가능한 금액이 얼마나 남은지 체크하는 가계부 어플

## Develop Environments
* Language: Typescript
* Framework: React native
* Database: sqlite
